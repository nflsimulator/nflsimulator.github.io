Basics of Java


So you want to learn java. You might be programming for the first time or you might already know another language. Why should I learn Java, you may ask yourself. Beause over 5 billion devices all over the world have some kind of Java program in them. If you know Java, you pretty much know the basics to many more languages. At first it may seem intimidating but it gets easier after you learn the basics. Java is a great language to learn for everybody (including beginners). So let's not waste any time and dive straight into the basics.

Data types



You've learned how to print text and the basics about curly brackets. Now it's time to learn about data types. A data type stores data in a name.

Ints and doubles



Ints and doubles are primitive data types that store numerical values. Ints can store any whole number while doubles store decimal numbers. You will get an error if you try to put a decimal in an int, but you will not get an error if you try to put an int in a double. The double automatically puts a .0 after the int. Booleans



The boolean might be the most simple data type. It either stores true or false (Nothing more). It might be useful if you're making a game and you have a boolean called "isalive" which is true if the character is alive and false when it isn't. Try creating booleans and printing them.
